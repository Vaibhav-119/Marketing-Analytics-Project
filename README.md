# 📊 Marketing Analytics Dashboard (Power BI | SQL | Python)

## 🔍 Overview
This project delivers an end-to-end **Marketing Analytics Solution** combining **SQL, Python, and Power BI** to analyze customer journeys, campaign performance, product conversions, and customer sentiment.

It covers the full data workflow:
**Data Cleaning (SQL) → Sentiment Analysis (Python) → Visualization (Power BI)**

---

## 🚀 Key Highlights
- 📈 Achieved **9.6% overall conversion rate**
- 👀 Analyzed **9M+ views, 1.78M clicks, 414K likes**
- ⭐ Average customer rating: **3.7 / 5**
- 🧠 Built **sentiment analysis model** using Python
- 🗄️ Performed **data cleaning & transformation using SQL**

---

## 📌 Problem Statement
Marketing teams lacked visibility into:
- Customer drop-offs in conversion funnel  
- Product-level performance  
- Social media engagement trends  
- Customer sentiment insights  

This project solves these by integrating data engineering, analytics, and visualization.

---
## 📊 Key Insights
- ⚠️ High traffic does not always lead to high conversions  
- 🎯 Certain niche products outperform general products  
- 📉 Engagement declines over time → possible content fatigue  
- 👍 Strong positive sentiment indicates good product-market fit  

---

## 💡 Business Recommendations
- Improve **landing pages & CTA placement** to reduce drop-offs  
- Allocate budget toward **high-converting products**  
- Refresh content strategy to maintain engagement  
- Use positive reviews for **social proof & retargeting**
